Material UI Flat colors theme


## Installation
Install using [npm](http://npmjs.com):
```sh
npm install @ox2/theme-mui --save
```
Install using [yarn](http://yarnpkg.com):
```sh
yarn add @ox2/theme-mui
```

## Usage
```js
import theme form '@ox2/theme-mui'

```
